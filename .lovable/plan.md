

# החלפת מנוע הקראה קולית ל-Web Speech API (דפדפן מובנה)

## הבעיה
קולות ElevenLabs לא מייצרים עברית תקינה למרות ניסיון עם מספר קולות שונים.

## הפתרון המוצע
במקום Google Cloud TTS (שדורש הגדרת מפתח API נוסף), נשתמש ב-**Web Speech API** -- יכולת מובנית בדפדפן שתומכת בעברית באופן מלא, בחינם, ובלי שום API key.

### יתרונות
- עובד בעברית באופן מקומי בדפדפן
- לא צריך מפתח API או שרת חיצוני
- מהיר -- אין המתנה לתשובה מרשת
- בחינם לחלוטין

### חסרונות
- הקול תלוי בדפדפן ובמערכת ההפעלה של המשתמש
- פחות "טבעי" מקול AI מתקדם

## שינויים טכניים

### 1. שכתוב `src/hooks/useTTS.ts`
- הסרת הקריאה ל-Edge Function של ElevenLabs
- שימוש ב-`window.speechSynthesis` עם בחירת קול עברי (`he-IL`)
- שמירה על אותו ממשק חיצוני (`speak`, `stop`, `isPlaying`, `isLoading`)

### 2. ללא שינוי ב-`Workout.tsx`
- כפתור הרמקול ישאר זהה -- רק המנוע מאחורי הקלעים משתנה

### 3. Edge Function `tts`
- אפשר להשאיר או למחוק -- לא ייעשה בו שימוש יותר
